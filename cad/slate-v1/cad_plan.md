# Slate v1 CAD Plan

## CAD scope for v1

The first Slate v1 CAD work focuses on packaging and dimensional coherence, not production-ready mechanical detail.

Scope includes:

- dimensional blocking model
- enclosure shell
- display slab
- PCB placeholder
- battery volume block
- vapor chamber block
- graphite spreader layer
- lateral camera prism path
- antenna keep-out zones

First CAD files are not production models. They are packaging models used to test whether the internal volume story is even coherent.

## Planned file outputs

- `enclosure.step`

- `pcb-placeholder.step`
